# 首页统计导出CSV字段说明

## 一、导出文件格式

- 文件名：`统计报表_{统计周期}.csv`
- 编码：UTF-8（带BOM头 `\uFEFF`）
- 分隔符：逗号 `,`

## 二、统计周期

| 类型 | 周期范围 | 示例 |
|------|---------|------|
| 本月 | 当月1日 ~ 当月最后一天 | 2025年7月 |
| 本年 | 当年1月1日 ~ 当年12月31日 | 2025年 |
| 自定义 | 用户选择的起止日期 | 2025-01-01 ~ 2025-06-30 |

## 三、CSV内容结构

### 第一段：概览指标

| 字段名 | 说明 | 数据来源 |
|--------|------|---------|
| 统计周期 | 当前选择的时间范围 | 用户选择的period |
| 入库总卷数 | 该周期内入库的档案总卷数 | `ingest`记录的`volumeCount`求和 |
| 入库批次 | 该周期内入库记录的条数 | `ingest.length` |
| 新增项目 | 当前馆藏中的项目数量 | `archiveData`中`tag=zj`的记录数 |
| 新增工程 | 当前馆藏中的工程数量 | `archiveData`中`tag=zj_item`的记录数 |
| 新增案卷 | 当前馆藏中的案卷数量 | `archiveData`中`tag=zj_volume`的记录数 |
| 新增文件 | 当前馆藏中的文件数量 | `archiveData`中`file_type=file`的记录数 |
| 利用总量 | 该周期内利用记录的总条数 | `utilization.length` |
| 无纸化出证率 | 下载方式占比 | `download`记录数 / 利用总量 ×100% |

### 第二段：载体类型统计

| 字段名 | 说明 |
|--------|------|
| 类型 | 载体类型名称 |
| 卷数 | 该载体类型的入库卷数 |

**载体类型枚举值：**
- 竣工图纸
- 文字卷宗
- 工程照片
- 光盘介质

### 第三段：利用人身份统计

| 字段名 | 说明 |
|--------|------|
| 身份 | 利用人身份类型 |
| 人次 | 该身份的利用人次 |

**身份枚举值：**
- 建设单位
- 施工单位
- 产权单位
- 律师
- 物业单位
- 房屋产权人
- 党政/纪检/司法机关
- 其他

### 第四段：利用目的统计

| 字段名 | 说明 |
|--------|------|
| 目的 | 利用目的类型 |
| 占比(%) | 该目的占总利用量的百分比 |

**目的枚举值：**
- 复印备案
- 领取房产证
- 生产建设
- 解决纠纷
- 查阅核对
- 调查取证
- 其他

## 五、新增文件统计说明

新增项目/工程/案卷/文件为**馆藏总量**（非周期内新增），数据来自`archiveData.json`的实时统计：
- 项目：`tag=zj`且`parent_id=0`的记录
- 工程：`tag=zj_item`的记录
- 案卷：`tag=zj_volume`的记录
- 文件：`file_type=file`的记录
