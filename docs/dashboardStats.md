# 首页统计模块 — 需求规格说明书

> 版本：v1.0

---

## 1. 功能定位

首页统计是用户进入系统后看到的**第一屏**，提供馆藏数据和档案利用数据的可视化概览。

---

## 2. 统计周期

位于页面顶部，全局控制所有图表数据的时间范围。

| 周期 | 说明 |
|------|------|
| 本月 | 当前自然月的数据 |
| 本年 | 当前自然年的数据 |
| 自定义 | 用户选择起止日期 |

右上角显示当前周期标签（如 `统计周期：2026年7月`），提供刷新按钮和一键导出功能。

---

## 3. Tab 切换

统计数据分为两个大类，通过 Tab 切换展示：

| Tab | 图标 | 颜色 |
|-----|------|------|
| 馆藏数据统计 | Database | 蓝色 |
| 档案利用统计 | TrendingUp | 绿色 |

默认显示「馆藏数据统计」。

---

## 4. 馆藏数据统计

### 4.1 顶部概况卡片（6 个）

| 指标 | 说明 |
|------|------|
| 入库总卷数 | 统计周期内入库的档案总卷数 |
| 入库批次 | 统计周期内的入库操作次数 |
| 新增项目 | 当前系统中的项目总数 |
| 新增工程 | 当前系统中的工程（单体）总数 |
| 新增案卷 | 当前系统中的案卷总数 |
| 新增文件 | 当前系统中的文件总数 |

> 「新增项目/工程/案卷/文件」为系统累计数据，不随统计周期变化。

### 4.2 图表区（4 个图表，2×2 网格）

| 位置 | 图表类型 | 标题 | 数据说明 |
|:----:|---------|------|---------|
| 左上 | 环形图 | 馆藏载体类型占比 | 按载体类型（竣工图纸、文字卷宗、工程照片、光盘介质）统计卷数占比 |
| 右上 | 折线图 | 入库趋势 | 按月份统计入库卷数变化 |
| 左下 | 横向柱状图 | 载体类型统计 | 11 种载体类型（文字、图纸、照片、底图、录音带、录像带、光盘、磁带、磁盘、缩微片、其他）的卷数分布 |
| 右下 | 柱状图 | 热门馆藏分类 | 按档案类型（工程档案、招投标档案、音像档案、村镇档案、文书档案）统计卷数 |

---

## 5. 档案利用统计

### 5.1 顶部概况卡片（2 个）

| 指标 | 说明 |
|------|------|
| 利用总量 | 统计周期内的利用总人次 |
| 无纸化出证率 | 下载人次占利用总人次的比例 |

### 5.2 图表区（3 个图表，动态排列）

| 图表类型 | 标题 | 数据说明 |
|---------|------|---------|
| 横向条形图 | 服务利用人结构占比 | 按 8 种利用人身份统计人次（建设单位、施工单位、产权单位、律师、物业单位、房屋产权人、党政/纪检/司法机关、其他） |
| 面积图 | 利用方式趋势 | 按月统计下载 vs 打印的利用量变化 |
| 环形图 | 利用目的分析 | 按 7 种利用目的统计占比（复印备案、领取房产证、生产建设、解决纠纷、查阅核对、调查取证、其他） |

---

## 6. 导出功能

点击「导出」按钮，按当前统计周期生成 CSV 文件：

- 文件编码：UTF-8 BOM（Excel 可直接打开）
- 包含数据：概况指标、载体类型统计、利用人身份统计、利用目的统计
- 文件名格式：`统计报表_{统计周期}.csv`

---

## 附录：数据来源

| 数据 | 来源 | 说明 |
|------|------|------|
| 入库记录 | 模拟数据生成器 | 2022年1月 ~ 2026年12月，每月 30~80 条 |
| 利用记录 | 模拟数据生成器 | 2022年1月 ~ 2026年12月，每月 100~300 条 |
| 项目/工程/案卷/文件数 | archiveData.json | 系统累计数据，不随周期变化 |
