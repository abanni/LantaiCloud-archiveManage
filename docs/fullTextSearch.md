# 全文检索模块 — 需求规格说明书

> 版本：v1.0

---

## 1. 功能定位

全文检索是用户对档案**正文内容**进行深度搜索的功能。区别于综合查询（仅搜索元数据），全文检索覆盖档案文件经 OCR 识别后的文本内容，并引入 AI 辅助增强搜索体验。

---

## 2. 搜索范围

### 2.1 层级勾选

| 层级 | 默认状态 |
|------|---------|
| 项目级 | ☑ 已勾选 |
| 工程级 | ☑ 已勾选 |
| 案卷级 | ☑ 已勾选 |
| 文件级 | ☑ 已勾选 |

取消勾选的层级不参与检索。

### 2.2 搜索维度

用户输入关键词后，同时在以下维度进行匹配：

| 维度 | 说明 |
|------|------|
| 元数据字段 | 各层级的标题、责任者、保管期限等字段（与综合查询一致） |
| 档案正文 | OCR 识别出的文件全文内容 |
| AI 摘要 | 预先生成的档案内容摘要 |

---

## 3. AI 深度搜索模式

页面右上角提供 AI 模式开关，默认开启。

### 3.1 开启 AI 模式

- 点击「检索」后，展示分步 AI 分析动画
- 处理步骤：意图分析 → 关键词提取 → 元数据匹配 → 正文搜索 → 摘要生成
- 延迟约 2 秒模拟 AI 处理过程
- 结果展示增加 AI 相关度评分和 AI 摘要

### 3.2 关闭 AI 模式

- 普通关键词搜索，无动画和延迟
- 结果仅展示匹配项，无相关度评分和 AI 摘要

---

## 4. 搜索结果展示

### 4.1 结果列表

每条结果包含以下信息：

| 元素 | 说明 |
|------|------|
| 层级徽标 | 项目级蓝 / 工程级紫 / 案卷级橙 / 文件级灰 |
| 标题 | 命中的档案名称 |
| AI 相关度 | 百分比评分（仅 AI 模式） |
| AI 摘要 | 紫色背景摘要区域（仅 AI 模式） |
| 上级路径 | 文件级结果显示所属案卷 → 工程路径 |
| 匹配段落 | 高亮关键词的原文片段 |

### 4.2 AI 处理动画

检索过程中展示：

```
🧠 AI 正在分析
意图分析 → 关键词提取 → 元数据匹配 → 正文搜索 → 摘要生成
```

步骤逐条推进，伴随脉动动画效果。

---

## 5. 预览面板

选中某条结果后，右侧展示预览面板：

| Tab | 内容 | 说明 |
|-----|------|------|
| 元数据 | 只读字段列表 | 显示该条记录的所有元数据字段 |
| 预览 | PDF 嵌入 | 仅文件级可用 |

AI 模式下，元数据面板额外展示 AI 摘要和正文原文摘要。

---

## 6. 利用清单

- 每条结果可勾选加入利用清单
- 预览面板底部提供「加入利用清单」按钮
- 已加入条目显示 ✓ 状态

---

## 附录：数据来源

| 数据 | 说明 |
|------|------|
| 元数据 | archiveData.json（与综合查询同源） |
| 全文索引 | fullTextIndex.json，预生成的 1253 条 OCR 示例文本 |
| AI 摘要 | 预生成的档案内容摘要 |
| 相关度评分 | 基于关键词命中数量和位置预先计算的置信度 |
